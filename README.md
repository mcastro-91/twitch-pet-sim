# Twitch Pet SIM Overlay

Mascota virtual tipo Tamagotchi para OBS + Twitch Chat.

## Comandos

- !feed → +hambre
- !play → +diversión
- !heal → +salud
- !sleep → descanso leve

## Uso en OBS

1. Añadir Browser Source
2. URL → index.html
3. Activar transparencia

## Persistencia

Se guarda automáticamente en localStorage.
